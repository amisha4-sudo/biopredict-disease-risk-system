# biopredict-disease-risk-system
BioPredict is a machine learning-based healthcare system that predicts disease risk based on patient data, enabling early diagnosis and preventive care.


## Features
- Disease risk prediction  
- ML model (Random Forest)  
- Interactive input interface  

## Tech Stack
- Python  
- Scikit-learn  
- Streamlit  

## Installation
pip install -r requirements.txt

## Run
streamlit run app.py

## How It Works
- Takes patient features as input  
- Uses trained ML model  
- Predicts disease risk  

## Future Improvements
- Add multiple diseases  
- Explainable AI (SHAP)
